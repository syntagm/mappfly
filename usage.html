Usage

The latest jLocator files:

  • JS files can be found in the '/minified' folder in jLocator archive.
  • CSS files can be found in the '/html/css' folder in jLocator archive.

Add the following HTML to the HEAD section of your page: 

						
<!-- css -->				
<link href="css/jlocator.min.css" rel="stylesheet" type="text/css" />

<!-- jlocator is based on jplist plugin: http://jplist.no81no.com -->
<script src="js/jplist.min.js"></script>	

<!-- jlocator -->		
<script src="http://maps.google.com/maps/api/js?sensor=false&libraries=places,geometry"></script>
<script src="js/jlocator.min.js"></script>

<!-- for IE 7: json support and ie 7 style -->
<!--[if lt IE 8]>
<script src="js/json2.min.js"></script>
<![endif]-->

Add plugin call to the HEAD section of your page: 

						
<script type='text/javascript'>	
	$('document').ready(function(){				
		$('#jlocator').jlocator();
	});
</script>
