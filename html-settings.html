HTML Settings

Data Attribute Description
data-type="panel" jPList panel section.
jLocator uses jQuery jPList plugin for sorting, filtering and pagination.
You can find jPList documentation here.
data-type="controls" Controls section in jPList panel.
Used for controls like sort dropdown, autocomplete etc.
data-type="stores" Stores list data type
data-type="store" One store data type
data-type="no-results" 'No results' section
data-type="map" Map data type
