Button Text Filter Group Control

All Controls

HTML Settings:

Button text filter control is a toggle filter.
When the button is selected, only elements that contain its text value are shown (data-text attribute).
When the button is not selected, all elements are shown.
In this control any html element can be used: SPAN, INPUT, DIV, etc.
OR logic is used inside each group, AND logic between different groups

Data Attribute Description Values Example
data-path jQuery path to the items jQuery path data-path=".title"
data-text filter text value any text value data-text="architecture"
data-button declare a filter button true or false data-button="true

HTML Layout: 

						
<div
	class="filter-buttons-group"
	data-control-type="button-text-filter-group"
	data-control-action="filter"
	data-control-name="button-text-filter-group-1">

	<span
		class="filter-button-span"
		data-path=".architecture"
		data-text="architecture"
		data-button="true">Architecture</span>
	
	<span
		class="filter-button-span"
		data-path=".christmas"
		data-text="christmas"
		data-button="true">Christmas</span>
</div>

JavaScript Settings: 

						
<script type="text/javascript">	
	$('document').ready(function(){
	
		$('#jlocator').jlocator({
		
			....
			
			,jplist:{
			
				//panel controls
				control_types: {
					
					'button-text-filter-group':{
						class_name: 'ButtonTextFilterGroup'
						,options: {
							ignore: '[~!@#$%^&*()+=`\'"\/\\_]+' //[^a-zA-Z0-9]+ not letters/numbers: [~!@#$%^&*\(\)+=`\'"\/\\_]+
						}
					}
				}
			}			
			
		});
	});
</script>
