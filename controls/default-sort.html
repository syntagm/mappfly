Default Sort Control

All Controls

'Default sort' control used instead of 'sort dropdown' control to define the inital sort order.

HTML Settings:

Data Attribute Description Values Example
data-path jQuery path to the items jQuery path or "default" for the default/initial sorting data-path=".title"
data-path="default"
data-order ascending or descending sorting "asc" or "desc" data-order="asc"
data-order="desc"
data-type sort data type "text", "number" or "datetime" data-type="text"
data-datetime-format datetime format The following wilcards could be used: {year}, {month}, {day}, {hour}, {min}, {sec}. data-datetime-format="{month}/{day}/{year}"
data-ignore regular expression which allows remove characters before sorting (for example, spaces, dashes etc.) regular expression data-ignore="[^a-zA-Z0-9]+"

HTML Layout: 

						
<!-- default sort hidden control -->
<div 
	class="hidden" 
	data-control-type="default-sort" 
	data-control-name="sort" 
	data-control-action="sort"
	
	data-path=".title" 
	data-order="desc" 
	data-type="text">
</div>

JavaScript Settings: 

						
<script type="text/javascript">	
	$('document').ready(function(){
	
		$('#jlocator').jlocator({
		
			....
			
			,jplist:{
			
				...
				
				//panel controls
				,control_types: {
					
					'default-sort':{
						class_name: 'DefaultSort'
						,options: {}
					}
				}
			}			
			
		});
	});
</script>
