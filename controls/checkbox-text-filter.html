Checkbox Text Filter Control

All Controls

Checkbox text filter control is used for text search within the keywords element (it could be any HTML element like siv, span, p, etc). It searches for a text from selected checkboxes (their values). It supports AND/OR logic for the selected checkboxes.

HTML Settings:

Data Attribute Description Values Example
data-path jQuery path to the keywords container jQuery path data-path=".keywords"
data-logic If Logical OR is used: the item is shown if at least one keyword is contained in the keywords list.
If Logical AND is used: the item is shown if all keywords are contained in the keywords list. 		"or" / "and" data-logic="or"

HTML Layout: 

						
<!-- checkbox text filter -->
<div
class="cb-text-filter text-shadow"
data-control-type="checkbox-text-filter"
data-control-action="filter"
data-control-name="keywords"
data-path=".keywords"
data-logic="or"> <!-- "or" / "and"  -->

    <div class="cb">
        <input
            value="architecture"
            id="architecture"
            type="checkbox"
        />

        <label for="architecture">Architecture</label>
    </div>

    <div class="cb">
        <input
            value="christmas"
            id="christmas"
            type="checkbox"
        />

        <label for="christmas">Christmas</label>
    </div>
</div>

JavaScript Settings: 

						
<script type="text/javascript">
	$('document').ready(function(){

		$('#jlocator').jlocator({
		
			....
			
			,jplist:{
			
				...
				
				//panel controls
				,control_types: {
					
					'checkbox-text-filter':{
						class_name: 'CheckboxTextFilter'
						,options: {
							ignore: '' //regex for the characters to ignore, for example: [^a-zA-Z0-9]+
						}
					}
				}
			}			
			
		});
	});
</script>
