Source Code

Where to find:

jLocator source code can be found in the 'source-code/jlocator' folder in jLocator archive.
Please check out jLocator API for more details.

How to minify:

For the source code minimization and compilation any tool can be used.
For example, Google Closure Compiler or UglifyJS.

How to use sources directly:

To use jLocator sources directly please include these js files instead of jlocator.min.js: 

						
<script src="../source-code/jlocator/jlocator.js"></script>
<script src="../source-code/jlocator/controller/controller.js"></script>
<script src="../source-code/jlocator/view/panel.js"></script>
<script src="../source-code/jlocator/view/map.js"></script>
<script src="../source-code/jlocator/models/store.js"></script>
<script src="../source-code/jlocator/controls/autocomplete.js"></script>
<script src="../source-code/jlocator/controls/autocomplete-radius.js"></script>
